Project Euler problem 009

A Pythagorean triplet is a set of three natural numbers, <b<c< b < c<b<c, for which,

a2+b2=c2 a^2 + b^2 = c^2 a2+b2=c2

For example, 32+42=523^2 + 4^2 = 5^232+42=52.

There exists exactly one Pythagorean triplet for which a+b+c=1000a + b + c = 1000a+b+c=1000.

Find the product a⋅b⋅ca \cdot b \cdot ca⋅b⋅c.

Hands-on

Mi first approach was to find aaa and bbb such that the square of both numbers summed up to the ccc squared. I achieved it by providing the desired value of ccc to find the triplet corresponding to that value.

Because a<b<ca < b < ca<b<c, it is possible to allow a to start at a=1a=1a=1, as a consequence b=2b=2b=2, thus making the minimum value c can take equal to c=3c=3c=3.

;; a: adjacent side
;; b: opposite side
;; c: hypotenuse
(defun find-triplet-aux (a b c &optional (a2 (expt a 2)) (b2 (expt b 2)) (c2 (expt c 2)))
  ;(format t "~a~%" (list a b))
  (cond ((equalp c2 (+ a2 b2)) (list a b c))
        ((< b c) (find-triplet-aux a (1+ b) c a2 (expt (1+ b) 2)))
        ((< a (- c 2)) (find-triplet-aux (1+ a) (+ 2 a) c (expt (1+ a) 2) (expt (+ 2 a) 2)))
        (t nil)))

(defun find-triplet (c)
  (find-triplet-aux 1 2 c))

(defun sum/lst (lst)

Round 2

I searched for some ideas on the internet, and came up with a simple solution.

(defun triplet-of-n (n &optional (a 1) (b 2) (blim (1- (floor n 2))))
  (let ((c (- n a b)))
    ;(format t "~a~%" (list a b c))
    (cond ((is-triplet? a b c) (list a b c))
          ((< a (1- b)) (triplet-of-n n (1+ a) b))
          ((< b blim) (triplet-of-n n 1 (1+ b))))))

It is an iterative solution where the value of ccc is computed as c=n−a−bc = n - a - bc=n−a−b. Instead of trying to find the triplet (a,b,c)(a,b,c)(a,b,c) that sums up to NNN, it is used to compute the value that ccc should have. Then, it is asked if the current (a,b,c)(a,b,c)(a,b,c) is indeed a Pythagorean triplet with is-triplet?.

(defun is-triplet? (a b c)
  (when (equalp (+ (expt a 2) (expt b 2)) (expt c 2)) t))

If (a,b,c)(a,b,c)(a,b,c) is a Pythagorean triplet, the product a⋅b⋅ca \cdot b \cdot ca⋅b⋅c is easily obtained with triplet-prod.

(defun triplet-prod (triplet)
  (when triplet (prod/lst triplet)))

These functions use when, meaning that only in the case that the'll only evaluate if the condition is not nil. Lastly, the function prod/lst is based on reduce just as sum/lst.

(defun prod/lst (lst)
  (reduce #'(lambda (acc x) (* acc x)) lst :initial-value 1))

Lastly, I didn't explained why function triplet-of-n has an optional argument that sets the upper limit of bbb set to ⌊N/2⌋\lfloor N / 2 \rfloor⌊N/2⌋, so allow me to tackle this point. First, we have to keep in mind that a<b<ca < b < ca<b<c, it is easy to deduce that the minimum value that aaa can take is a=1a=1a=1. Because of this, the smallest bbb value is b=2b=2b=2. Similarly, the smallest value ccc can take is c=3c=3c=3.

What about the maximum values? I only assumed the scenario where a=1a=1a=1 to find an answer to this question. Assuming this is the case, then

a+b+c=N1+b+c=Nb′+c=N. \begin{align*} a + b + c & = N \\ 1+ b + c & = N \\ b' + c & = N . \end{align*} a+b+c1+b+cb′+c​=N=N=N.​

The constraint b<cb < cb<c doesn't apply here, in fact, it is allowed that b′≤cb' \leq cb′≤c. If both are equal, b′=cb' = cb′=c, that means that c=b′=⌊N/2⌋c = b'= \lfloor N/2 \rfloorc=b′=⌊N/2⌋. Therefore,

b+1=b′b+1=⌊N/2⌋b=⌊N/2⌋−1. \begin{align*} b + 1 & = b' \\ b + 1 & = \lfloor N/2 \rfloor \\ b & = \lfloor N/2 \rfloor - 1 . \end{align*} b+1b+1b​=b′=⌊N/2⌋=⌊N/2⌋−1.​

Now we have the limits of interest, the lower limit of aaa and the lower and upper limits of bbb. I didn't tried to find an upper limit for aaa because I didn't needed it for my implementation. Likewise, I didn't use the upper limit of ccc in the implementation of triplet-of-n, only to find the upper limit of bbb.

triplet-of-n starts at (a,b)=(1,2)(a,b) = (1,2)(a,b)=(1,2), it doesn't adds 1 unit to aaa because that violates the condition a<ba < ba<b. So, it adds 1 to bbb and sets a←1a \gets 1a←1, making the next pair of values are (a,b)=(1,3)(a,b)=(1,3)(a,b)=(1,3), and the second pair of values equal to (a,b)=(2,3)(a,b) = (2,3)(a,b)=(2,3). This process continues until a triplet is found or (a,b)=(⌊N⌋−2,⌊N⌋−1)(a,b) = (\lfloor N \rfloor - 2, \lfloor N \rfloor - 1)(a,b)=(⌊N⌋−2,⌊N⌋−1).
